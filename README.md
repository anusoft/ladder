ladder
======

顾名思义，作用你懂的！

在UNIX like系统，包括Linux、Mac OS X甚至Cygwin，Terminal中运行命令：

* mkdir ladder
* cd ladder
* curl -L http://git.io/lddr | sh
* ./start.sh

注意，运行上面的命令前，需要确保你的系统中已经安装有基本工具curl，wget，bash，依赖库OpenSSL开发包，以及开发工具gnu make，gcc等。
